`catalog.tsv` contains synthetic catalog data generated with the following parameters

- `t_pre = 0`
- `t_begin = 0`
- `t_end = 1000`
- $m_{\mathrm{r}} = 6$
- $\mu_{\mathrm{es}} = 0.1$
- $K_{\mathrm{es}} = 10$
- $c = 0.01$
- $\alpha = 1.1$
- $p = 1.1$

```bash
bin/etas_solve.sh \
   --t_normalize_len=1 \
   --m_for_K=6 \
   --t_pre=0 \
   --t_begin=0 \
   --t_end=1000 \
   --data_file=example/etas_solve/catalog.tsv |
   release/bin/etas_solve.exe
```
